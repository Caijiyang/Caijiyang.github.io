QueryRunner类结果处理的八种方法

发表于 2020-10-04 更新于 2023-07-12 分类于 1 后端 ， 1.1 Java

:fleur_de_lis: 在连接数据库的时候(MySQL)时，会遇到的问题：QueryRunner类结果处理的八种方法

QueryRunner类结果集处理的八种方法

调用QueryRunner的方法是：

MapListHandler

将结果集每一行存储到Map集合中，键：列名；值：数据

MapHandler

将结果集的第一行数据，封存到Map集合中

ScalarHandler

查询后，只有一个结果

ColumnListHandler

结果集，指定列的数据存储到List集合中，

List每个列数据类型不同。

BeanListHandler

结果集每一行数据，封装JavaBean对象

多个JavaBean对象，存储到List集合

BeanHandler

将结果集的第一行数据,封装成JavaBean对象

ArrayListHandler

将结果集的每一行,封装到对象数组中

ArrayHandler

将结果集的第一行存储到对象数组中 Object[]
